Scan-line-Fill
==========

Implemenation of Polygon Fill using the Scanline algorithm using OpenGL and GLUT. 

![](http://i.imgur.com/IRC2iDA.gifv)

#Use

Simply right-click where you want your verticies, and then left-click -> "Draw Polygon" to fill. 

#Dependencies

GLUT and OpenGL.
